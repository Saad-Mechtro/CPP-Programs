# CPP-Programs

This repository contains basic C++ programming tasks implemented using functions, arrays, strings, and matrices.

## Programs Included

1. Finding Maximum and Minimum Values in an Array  
2. Standard Deviation Calculation  
3. 3x3 Matrix Multiplication Using Functions  
4. String Analysis Program  
   - Total Characters  
   - Spaces  
   - Tabs  
   - Lines  
5. Mean, Median, and Mode Calculation  

## Features
- Separate Functions
- Array Passing in Functions
- Simple and Beginner Friendly Logic
- Clean and Organized Code
- Step by Step Implementation

## Language Used
C++
